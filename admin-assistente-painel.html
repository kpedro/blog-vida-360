Painel Planejador Estúdio Ver blog

Tire dúvidas sobre sequência de posts, categorias, calendário e coerência da marca. IA padrão — respostas mais curtas e rápidas (modelo configurável por secret BLOG360_ADMIN_GEMINI_MODEL_PADRAO, padrão gemini-2.0-flash). IA dedicada — orientação mais profunda (secret BLOG360_ADMIN_GEMINI_MODEL_DEDICADO, padrão gemini-2.5-flash). Requer deploy da Edge Function blog-admin-assistant e secret GEMINI_API no Supabase. Copiar em cada mensagem ou Copiar conversa para colar no Estúdio / n8n / outro prompt. O histórico fica neste navegador (localStorage) e actualiza ao receber cada resposta. Pode colar prints (Ctrl+V) na caixa de texto ou usar Anexar print — a IA analisa a imagem (requer redeploy da função blog-admin-assistant). Quando a ideia estiver madura, use Afinar no Estúdio: leva a conversa (texto) para o Montar com assistente do Estúdio, que transforma o contexto num prompt pronto para gerar conteúdo.

Histórico (neste navegador)

    «Abrir» substitui a conversa actual. «Estúdio» envia essa conversa ao assistente de prompt do Estúdio. «Apagar» remove só esta entrada local.

    Ou cole um print com Ctrl+V na caixa abaixo.

    Use Enter para enviar; Shift+Enter quebra linha. Abra esta página via http://localhost (npm run dev), não file://.
