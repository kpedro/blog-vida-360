Gere textos para landing, redes (com foco em legenda para Instagram) e artigos, alinhados ao tom do blog. Teste local: na pasta do projeto rode npm run dev (porta 5174; se precisar da 8080 use npm run dev:8080) e abra http://localhost:5174/admin-estudio-conteudo.html — abrir o HTML direto como file:// impede o navegador de chamar o Supabase. Proporção 4:5 e descarregar a imagem são experimentais. Após alterar o backend, faça deploy das funções generate-blog-studio-content e generate-blog-studio-image no Supabase.

Resultado

🔽 Google

Google (mesmo backend do PedagoFlow): com a conta ligada em PedagoFlow → Gestão → Integrações Google (mesmo projeto Supabase), «Google Docs» cria o ficheiro pela API e abre o documento. Sem ligação, copia-se o texto sem Markdown e abre-se docs.new para colar (Ctrl+V). «Conectar conta Google» inicia o mesmo OAuth do PedagoFlow (após autorizar pode redirecionar para o app PedagoFlow). Gmail usa composição web; textos longos podem ser encurtados.

Imagem (opcional)

Usa a sugestão automática após gerar texto, ou descreva abaixo. Para Instagram em formato retrato clássico do feed, experimente 4:5.

Se você gerou a imagem acima (data URL), redes sociais não mostram preview. Envie o arquivo para um local público (ex.: Storage do Supabase) e cole aqui o link https direto do arquivo. Também pode usar só este link e manter a geração apenas como referência visual.

Texto sobre a imagem (capa estilo notícia)

Depois de gerar a imagem acima, use campos semelhantes ao Estúdio PedagoFlow: categoria com faixa amarela, manchete em destaque e rodapé (marca ou URL). O fundo escurece para a leitura. Gere a pré-visualização e descarregue a capa final para Instagram ou WhatsApp.
