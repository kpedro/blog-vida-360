Banner lateral na home e faixa no topo (todas as páginas onde o script está incluído: início, lista de artigos, post). Rode supabase/BLOG360_SITE_SETTINGS.sql e, se a tabela já existia antes, supabase/BLOG360_SITE_SETTINGS_FAIXA_OPORTUNIDADE.sql (ou rode o SQL completo de novo — os ADD COLUMN IF NOT EXISTS são seguros).

Pode ser link do Supabase Storage ou CDN. Proporção larga: ~1200×400.

Cole o mesmo URL que abre na barra do navegador. Ao salvar, removemos um segmento final duplicado (ex.: …/60338539204/60338539204) se for cópia errada.

Prévia

Faixa de oportunidade (topo do site)

Estilo “announcement bar” usado em blogs e lojas: uma linha chamativa abaixo do menu, com texto curto e botão opcional (ex.: prospectar distribuidores, parceria Amway ou outra empresa). Evitamos flash rápido; há apenas um brilho suave na borda para chamar atenção sem prejudicar acessibilidade.

Uma ou duas frases. Em telas pequenas o texto quebra em várias linhas.

Se vazio, só a mensagem aparece (sem botão).
